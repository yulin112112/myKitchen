一、核心世界观设定
「庖丁纪」宇宙法则
1.空间维度：
明界：人类可见的现实厨房
暗界：厨具觉醒后的"味境江湖"（只有凌晨3:15-4:44可见）

2.能量体系：
内力=「烟火气」：通过烹饪时食材与火焰交互产生
武功秘籍=《中华食经》残卷（记载108道失传菜式）

3.时空规则：
厨具损伤会影响对应武侠角色战力
每月十五食材新鲜度翻倍（满月BUFF）
